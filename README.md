### Miscellaneous

If you want more,

You can clone the project:

```
git clone https://github.com/mrdoomy/awesomehooks.git
```

Install dependencies:

```
npm install
```

Run all unit tests:

```
npm run test
```

And finally compile the project:

```
npm run build
```
